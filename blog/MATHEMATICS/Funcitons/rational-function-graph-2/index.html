식보다는 예제를 보는 것이 이해하기 좋습니다.

f(x)=x2+1xf(x) = \frac{x^2+1}{x}f(x)=xx2+1​ 이러한 식이 있다고 하죠,

f(x)=x2+1x=x2x+1x=x+1xf(x) = \frac{x^2+1}{x} = \frac{x^2}{x} + \frac{1}{x} = x + \frac{1}{x} f(x)=xx2+1​=xx2​+x1​=x+x1​

각 항을 각각의 함수로 보면 일차함수와 유리함수 하나씩으로 나눠집니다.

일차함수의 그래프 h(x) #

h(x)=xh(x) = x h(x)=x

유리함수의 그래프 g(x) #

g(x)=1xg(x) = \frac{1}{x} g(x)=x1​

이 둘을 더한 함수는 어떤 그래프일까요? #

f(x)=h(x)+g(x)f(x) = h(x) + g(x) f(x)=h(x)+g(x)

곂쳐서 보면 보이는 #

함수 f(x)=h(x)+g(x)f(x) = h(x) + g(x)f(x)=h(x)+g(x)는 유리함수 그래프 g(x)g(x)g(x)의 점근선이 일차함수 h(x)h(x)h(x) 를 따라가고 있다는 것을 알 수 있습니다.
