forall #

∀\forall∀: \forall

∀x,p\forall{x}, p∀x,p : x가 될 수 있는 모든 경우에 대해 조건 p가 만족된다.

∀x∈N,x>0\forall{x}\in \mathbb{N}, x > 0∀x∈N,x>0 = 모든 자연수에 속하는 x는 0 보다 크다.

∀x∈[0,∞),x≥0\forall{x} \in [0,\infin), \sqrt{x} \geq 0∀x∈[0,∞),x​≥0 = 모든 0 이상의 실수에 대해 x의 제곱근은 0 보다 크거나 같다.

∀x except x≠0,x⋅1x=0\forall{x} \ except \ x \neq 0, x \cdot \frac{1}{x} = 0∀x except x=0,x⋅x1​=0 = 모든 실수 x에 (0은 제외), x⋅1x=0x \cdot \frac{1}{x} = 0x⋅x1​=0을 만족한다.

exist #

∃\exist∃: \exist

∃x,x−1=0\exist{x}, x - 1 = 0∃x,x−1=0 x−1=0x - 1 = 0x−1=0을 만족하는 x가 적어도 하나는 존재한다.

∀x,∃a,x+a=x\forall{x}, \exist{a}, x + a = x∀x,∃a,x+a=x = 모든 x에 대해 x+a=xx+a=xx+a=x를 만족하는 a가 적어도 하나는 존재한다.

∀x,∃a,x+a=x⟶a=0\forall{x}, \exist{a}, x + a = x \longrightarrow a = 0∀x,∃a,x+a=x⟶a=0 = 모든 x에 대해 x+a=xx+a=xx+a=x를 만족하는 a가 적어도 하나는 존재한다. 그 a는 0이다.

⊕\oplus⊕: 두 인자가 서로 다른 경우 true를 반환합니다.

false⊕false=truefalse \oplus false = truefalse⊕false=true

false⊕true=truefalse \oplus true = truefalse⊕true=true
