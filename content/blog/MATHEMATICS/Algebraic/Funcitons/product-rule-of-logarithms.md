# 성질

$log_b(x) + log_b(y) = log_b(xy)$

## 증명

$log_2(8) = 3$
$log_2(4) = 2$

위 두 식을 곱해 $log_2(8x4)$라고 한다면,
$log_2(8x4) = log_2(32) = 5$입니다.

$log_b(xy) = log_b(x) + log_b(y)$가 성립하는지 대입해보겠습니다.
$log_b(32) = log_b(8) + log_b(4) = 3 + 2$

잘 대입됩니다.
