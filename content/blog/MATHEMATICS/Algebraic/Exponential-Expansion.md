# 지수의 확장
- <a href="/blog/MATHEMATICS/Algebraic/Exponential-Expansion/">지수의 확장: 음의 지수 ✨</a>


## $10^{-1} = \frac{1}{10}$

- 지수가 자연수
    1. $10^0 = 10$
    2. $10^2 = 100$
    3. $10^3 = 1000$
    4. ...

- 지수를 음수로

1. $10^{-1} = \frac{1}{10}$
2. $10^{-2} = \frac{1}{100}$
3. $10^{-3} = \frac{1}{1000}$

- 연속적으로 

1. $10^{3} = 1000$
2. $10^{2} = 100$
3. $10^{1} = 10$
4. $10^{0} = 1$
5. $10^{-1} = \frac{1}{10}$
6. $10^{-2} = \frac{1}{100}$
7. $10^{-3} = \frac{1}{1000}$
