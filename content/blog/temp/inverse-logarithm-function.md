# 로그함수의 역함수 찾기

역함수를 찾을 식을 세워보겠습니다.

> $log_2{8} = 3$

여기서 함수는 $f(x) = log_2{x}$입니다.

x값을 대입해보면 $f(8) = 3 = log_2{8}$이 출력됩니다.

역함수를 정의해보겠습니다.

$f(x) = log_2{x}$
$y = log_2{x}$

$x = log_2{y}$
- x와 y의 역할을 바꿉니다.

$x = log_2{y}$
- y를 주어진 x의 함수로 표현합니다.

$x=2^y$
- $log_2{y}$는 2를 y제곱하면 x가 되는 함수입니다.

따라서 역함수로 표기하면

$f^{-1}(y) = 2^y$입니다.
